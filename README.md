# 📈 National Stock Exchange of India Ltd Dataset

## 📌 Project Overview
This repository contains historical stock market data of National Stock Exchange of India Ltd in CSV format. The dataset can be used for financial analysis, data visualization, stock trend analysis, and machine learning projects.

## 📂 Dataset Information
- **File Name:** `National_Stock_Exchange_of_India_Ltd.csv`
- **File Format:** CSV
- **Data Type:** Stock Market Data

- Study price movement patterns
- Practice data analytics using real-world datasets
